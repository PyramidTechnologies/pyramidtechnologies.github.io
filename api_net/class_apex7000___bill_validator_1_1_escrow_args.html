int Apex7000_BillValidator.EscrowArgs.Index
get

Index of note that is in escrow position. Index is an integer between 1 and 7 inclusive. Denomination mapping depends on the Apex 7000 firmware but here is the USD map:

1: $1 2: $2 (not used) 3: $5 4: $10 5: $20 6: $50 7: $100
