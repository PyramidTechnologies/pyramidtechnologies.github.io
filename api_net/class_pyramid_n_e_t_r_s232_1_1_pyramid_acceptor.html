void PyramidNETRS232.PyramidAcceptor.PauseAcceptance ( )

Disables the bill acceptor within the time period defined by the poll rate. The poll rate (RS232Config.PollRate, default 50 ms) is the maximum time between poll packets from master to slave. This command does not disconnect the serial port. Use Close() for that effect.

This effectively tells the acceptor to stop accepting bill but keep reporting status. The acceptor's lights will turn off after this call takes effect.
